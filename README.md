# SUPER-MART-SALES-DATA-ANALYSIS
This project contains an interactive Power BI dashboard built using the Superstore dataset. It provides deep insights into sales performance, customer segments, product categories, shipping modes, and regional trends. The dashboard includes advanced visualizations such as donut charts, line charts, bar graphs, geo-maps, and KPI indicators.
